# prairie-dog-vacuum
Your farm is infested with adorable prairie dogs! Use your prairie dog vacuum to clear the area of prairie dogs. 

# Dependencies
- React 16
- PixiJS 4
- react-pixi-fiber

# Installation
1) Clone repo
2) Run npm install
3) Run npm start
4) Open your browser and navigate to localhost:3000
